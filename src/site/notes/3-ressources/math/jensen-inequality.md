# Jensen Inequality
### Definition Finite Form
For real convex function $\varphi$ and positive weights $a_{i}$
$$\varphi \left( \frac{\sum a_{i}x_{i}}{\sum a_{i}} \right) \leq \left( \frac{\sum a_{i} \varphi(x_{i})}{\sum a_{i}} \right)$$
