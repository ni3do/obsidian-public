# L-smooth
### Definition
$l: \mathbb{R}^d \rightarrow \mathbb{R}$ is $L$-smooth for $L >0$:
$$\lvert\lvert \nabla l(\theta) - \nabla l(\vartheta)\rvert\rvert \leq L \lvert\lvert \theta - \vartheta \rvert\rvert  \quad \forall \theta, \vartheta$$
